## 📋 **CONVERSION REFERENCE**

### **Common Tailwind → CSS Conversions**

```jsx
// Layout
"bg-white shadow-md rounded-lg p-6" → "card" + "card-body"
"flex items-center justify-between" → "flex items-center justify-between"
"grid grid-cols-4 gap-6" → "grid grid-cols-4"

// Text
"text-xl font-bold text-gray-900" → style or custom class
"text-sm text-gray-600" → style or custom class

// Buttons
"bg-primary-600 text-white px-4 py-2 rounded-lg" → "btn btn-primary btn-md"

// Inputs
"w-full px-4 py-2 border rounded-lg" → "input"

// Badges
"px-2 py-1 rounded-full text-xs bg-green-100 text-green-700" → "badge badge-success"

// Tables
"overflow-x-auto" → "table-container"
"min-w-full divide-y divide-gray-200" → "table"
```
